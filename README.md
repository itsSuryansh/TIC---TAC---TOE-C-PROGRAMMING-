# TIC---TAC---TOE-C-PROGRAMMING-

This repository contains a console-based Tic-Tac-Toe game implemented in the C programming language. It is designed for beginners practicing arrays, functions, conditional logic, and user input handling.

## Features
- Two-player (X vs O) gameplay
- 3x3 ASCII board rendering
- Valid move checking
- Win detection (rows, columns, diagonals)
- Draw detection when board is full
- Clean and modular code structure

## How to Compile
Make sure GCC or another C compiler is installed.

### Windows (MinGW)
gcc tictactoe.c -o tictactoe.exe

### Linux / macOS
gcc tictactoe.c -o tictactoe

## How to Run

### Windows
./tictactoe.exe

### Linux / macOS
./tictactoe
